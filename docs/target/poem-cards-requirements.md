# 诗词卡片 - 目标需求

**日期**：2026-06-09
**状态**：目标文档（Target Doc）

---

## 目标

为孩子制作古诗词复习卡片：把 `poems_index.json` 中的诗词做成 A4 双面打印模板，裁剪后得到实体卡片。

## 数据范围

- **源文件**：`poems_index.json`
- **诗词数量**：共 200 首（序号 1-200）
- **元数据字段**：序号、题目、作者/朝代、课本页码、正文

## 卡片规格

### 正面（诗词内容）

- 顶部：序号小字
- 中间：诗词正文
- **居中排版**，不留内部空白

### 背面（题目信息）

- 序号
- 题目（醒目大字）
- 作者/朝代
- 课本页码（如 P192）

## 设计要求

| 项目 | 要求 |
|------|------|
| 纸张 | A4 双面打印，极致缩小页边距 |
| 布局 | 8 张卡片/页（4列2行），卡片无缝拼接，一刀双切 |
| 字体 | **宋体** |
| 字号 | **大**（不费眼睛） |
| 排版 | **绝对居中、优美**（长诗句智能断句） |
| 裁剪 | **极致便利**（无多余边距，沿虚线切开即可） |
| 输出格式 | **XLSX**（用户可调整） |

## 用户验收标准

- [x] 字号够大，5米外能看清标题
- [x] A4 打印不留空白，最好不留边缘
- [x] 卡片裁剪方便，无缝拼接，横竖切开即分离
- [x] 宋体字体正确显示
- [x] 双面打印时正反面物理位置绝对对齐
