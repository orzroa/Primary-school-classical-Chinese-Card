# 古诗词打印卡片生成器 (Poem Cards Generator)

这是一个用于生成 A4 双面打印古诗词复习卡片的 Python 工具。通过运行脚本，可以自动将预置的古诗词数据生成精美的 Excel (XLSX) 打印模板。

## ✨ 特性

- **完美的物理对齐**：专为 A4 双面打印设计。通过超细颗粒度的 Excel 网格系统与水平/垂直绝对居中策略，彻底解决双面打印时纸张正反面错位的问题。
- **极致的纸张利用率**：无间隙（背靠背无缝拼接）排版设计，极大压榨了页边距（仅保留 0.1 英寸安全边距）。打印完成后，只需沿虚线“一刀两断”即可剥离所有卡片，极其省力。
- **智能长句断句**：针对儿童复习场景的大字号护眼要求，遇到字数较多的长诗句（如《七律·长征》）时，算法会自动根据中文标点（，、；。）将其智能切分为多行，并在卡片中央进行完美的绝对垂直/水平居中。
- **纯净的数据驱动**：收录了完整 148 首古诗词正文与源信息的 `poems_index.json` 数据源，完全与代码解耦，便于增删改查。

## 🚀 快速开始

### 1. 环境准备

确保你已安装 Python 3 环境，并且安装了 `openpyxl` 依赖：

```bash
pip install openpyxl
```

### 2. 生成卡片模板

运行 `generate_cards.py` 脚本。你只需要传入想要生成的古诗**起始序号**和**结束序号**即可。

**注意**：为确保页面排版不留残缺，请求生成的卡片总数必须是 **8 的倍数**（单页 A4 纸刚好容纳 4列×2行 共 8 张卡片）。

```bash
# 生成第 1 首到第 16 首的卡片（共跨 2 张双面 A4 纸）
python3 generate_cards.py 1 16
```

执行完毕后，当前目录下会瞬间输出对应的 Excel 打印文件，例如：`poem_cards_1-16.xlsx`。

### 3. 如何打印

1. 使用 Excel 打开生成的 `.xlsx` 文件。
2. 打印时，请务必选择 **“打印整个工作簿”**。
3. 勾选 **双面打印**，装订方向选择 **沿短边翻转**（A4 横向双面的标准配置）。
4. （脚本已自动锁死了所有的页面设置和 100% 缩放比例，无需任何人工调整）。
5. 打印出纸后，直接沿卡片虚线裁剪即可。

## 📂 项目结构

- `generate_cards.py`：核心排版生成脚本。
- `poems_index.json`：包含 148 首古诗词（序号、题目、作者、正文、课本页码）的全量 JSON 数据。
- `docs/`：项目的初始需求目标和底层技术重构的深度分析过程记录。
